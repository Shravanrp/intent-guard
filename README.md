# IntentGuard

A Goal-Based Browser Productivity Assistant built as a Chrome Extension using Manifest V3.

## Features

- Set a focus goal
- Detect distracting websites
- Full-screen warning overlay
- 10-second countdown
- Track distraction attempts
- Chrome Storage API
- Responsive popup UI

## Tech Stack

- JavaScript
- HTML
- CSS
- Chrome Extension Manifest V3

## Installation

1. Clone the repository
2. Open chrome://extensions
3. Enable Developer Mode
4. Click Load unpacked
5. Select the project folder
